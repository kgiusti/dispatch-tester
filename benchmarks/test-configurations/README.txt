This directory contains directories containing configuration files for
some example deployments:

o) single-hop: standalone router

o) two-hop: two interconnected routers

o) router-qpidd: router front end to qpidd broker

o) router-artemis: router front end to artemis

o) router-fake-broker: router front end to another router acting as a
   broker (no route protocol running)

o) simple-edge: two edge routers connected via a single interior
   router


Unless otherwise noted all clients should send/receive from the
address "benchmark".
